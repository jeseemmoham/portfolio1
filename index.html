HireFlow - Smart Job Portal & ATS

A MERN stack–based smart job portal and applicant tracking system that connects recruiters and job seekers through secure authentication, role-based access, job posting, resume uploads, and real-time application tracking.

MongoDB Express.js React.js Node.js JWT Auth REST API

HireFlow is a full-stack web application built using the MERN stack that streamlines the recruitment process for companies and job seekers. The platform provides a centralized system where recruiters can post jobs, manage applications, and track candidate status, while job seekers can search and apply for jobs efficiently.

The system supports role-based access control with three user roles: Admin, Recruiter, and Job Seeker. Recruiters must be approved by the Admin before posting jobs, ensuring authenticity and platform trust. Job seekers can upload resumes, apply for jobs, and track application status in real time.

HireFlow includes features such as secure authentication using JWT, resume upload and parsing, advanced job search with filters, application status tracking, and dashboard analytics for recruiters and admins. Email notifications keep users informed about application updates and approvals.

This project is designed to reflect real-world enterprise recruitment systems, demonstrating strong backend architecture, secure APIs, and scalable full-stack development practices.
